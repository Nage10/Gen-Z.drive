# Gen-Z Drive

A secure, fast, and feature-rich file storage and collaboration platform built with PHP and React. Similar to Google Drive and NextCloud.

## Features

### Core Features
- 📁 File Upload/Download (Drag & Drop support)
- 📂 Folder Management
- 🔐 Secure User Authentication (JWT)
- 🔑 Two-Factor Authentication (2FA)
- 👥 File Sharing & Advanced Permissions
- 📜 Version History & Restore
- 🔍 Full-Text Search
- 👀 File Preview (Images, Videos, Documents)
- 🗑️ Trash & Restore
- 📊 Activity Logs

### Extra Features
- 🔄 Real-time File Sync
- 💬 File Comments & Collaboration
- 🔒 End-to-End Encryption
- 🎨 Dark Mode Support
- 📱 Mobile-Friendly UI
- ⚙️ Admin Dashboard
- 📈 Quota & Storage Management
- 🚀 REST API
- 🔐 SSL/HTTPS Ready
- 📧 Email Notifications
- ⭐ Favorites/Starred Files
- 🏷️ File Tags & Categories
- 📅 Calendar Integration
- 👤 User Profiles & Settings
- 🌍 Multi-Language Support

## Tech Stack

- **Backend**: PHP 8+, MySQL 8+
- **Frontend**: React 18+, Vite
- **Database**: MySQL with Indexing
- **Security**: JWT, OAuth2, Encryption
- **Server**: Apache
- **API**: RESTful API
